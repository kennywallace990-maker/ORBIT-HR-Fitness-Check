# ORBIT Program KB

This folder is the shared, program-level knowledge base for ORBIT. It is meant to hold MCP-friendly markdown artifacts that give agents reusable context across products.

## What belongs here
- Cross-product program context
- Enterprise strategy and FY planning context
- Shared operating model, governance, architecture, and roadmap context
- Synthesized markdown KB files with front matter that retrieval pipelines can ingest

## Folder conventions
- Root of this folder: MCP-ready knowledge base files in `.md`
- `Raw Files/`: source documents, decks, PDFs, images, and other supporting artifacts

## Retrieval guidance for agents
- Start here when the question is about ORBIT-wide context, enterprise priorities, or shared program assumptions.
- Use product-specific KB folders inside each product area for product-level details.
- Treat synthesized KB files as working context unless a source document or system-of-record confirms the latest state.
