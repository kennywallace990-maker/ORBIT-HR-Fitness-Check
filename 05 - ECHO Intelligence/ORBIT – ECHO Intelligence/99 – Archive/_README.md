## Archive Policy

When a document in any ORBIT folder is superseded by a new version:

1. Move the old version into the appropriate quarterly subfolder (e.g., `2026-Q2/`)
2. Rename with a date suffix if needed (e.g., `PRD, ECHO Intelligence (2026-03-17).md`)
3. Update the active document in its original folder
