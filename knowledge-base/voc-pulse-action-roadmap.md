## What This Source Is

This SharePoint folder is the post-2025 VOC Pulse action-planning hub for Core FC Team Member Experience work. It ties the 2025 VOC Pulse findings to 2026 action workstreams, project charters, trackers, playbooks, SOPs, and pilot artifacts.

For HR Fitness Check, this is not a primary line-item scoring source. It is valuable because it shows what the business is doing after VOC and TM Experience signals identify gaps. It can help future ORBIT reports explain available remediation paths, avoid generic recommendations, and point HR leaders toward already-sponsored workstreams.

Recommended product framing:

- Treat as downstream action-loop evidence.
- Treat as a seed for RM-005, the recommended interventions library.
- Link low Fitness Check or VOC-adjacent findings to approved playbooks only after governance confirms each artifact's approval status and audience.
- Do not ingest raw comments, named owner rosters, or action notes into ORBIT outputs unless classification and access rules explicitly allow it.

## VOC Pulse Baseline Context

The baseline report describes 19,058 2025 Team Member listening signals across 13 active Core FC sites and five listening channels.

Listening channel mix:

| Channel | Signal volume / share | Notes |
|---|---:|---|
| Public and unfiltered | 13,766 / 72.2% | Mostly VOC Board signals. |
| Leader-mediated formal | 1,399 / 7.3% | Standups. |
| Leader-mediated conversational | 2,660 / 14.0% | Informal conversations, Gemba walks, 1:1s, TM Experience Walks, Site Leadership Walks, and Sr. Team Engagements. |
| Structured roundtables | 449 / 2.4% | Roundtable feedback. |
| New-hire lifecycle surveys | 784 / 4.1% | New Hire survey lifecycle signals. |

## How Future Reports Should Use This Source

Use this source to answer questions like:

- If a site has weak VOC Board Management or CAT follow-up, are there already approved action workstreams the site can reuse?
- If a Fitness Check item is red, what non-generic action-plan starter can ORBIT offer without inventing a solution?
- If site-level VOC signals are high in a known cluster, is there a 2026 roadmap workstream already addressing that theme?
- Which recommendations should be phrased as "review the existing playbook/charter" rather than "create a new process"?
- Which follow-up metrics could validate whether action plans changed the Team Member experience?

Do not use this source to:

- Score a Standard Work item without a mapped measurement rule.
- Infer individual accountability.
- Publish named owners or raw action notes into an ORBIT summary.
- Treat a draft charter as approved guidance without owner/governance confirmation.
- Treat 2025 site clusters as current-state truth without validating the current quarter.

## Knowledge Base And Ingestion Implications

1. Add a new source family to the HR Fitness Check ontology: `voc_action_roadmap`.
2. Treat the roadmap as a recommendation and action-plan context layer, not as a rating engine input.
3. Create an approved-intervention registry if RM-005 moves forward. Minimum fields should include:
   - `intervention_id`
   - `source_workstream`
   - `source_artifact_url`
   - `approved_audience`
   - `approval_status`
   - `related_fitness_check_items`
   - `related_voc_themes`
   - `recommended_when`
   - `do_not_use_when`
   - `owner_group`
   - `last_reviewed_date`
4. Link future recommendation prompts to approved artifacts by URL rather than copying source content.
5. If action completion status exists in the workstream trackers, map it separately from the Fitness Check score. Action completion answers "what was done after the signal," not "was the stated HR process followed."
