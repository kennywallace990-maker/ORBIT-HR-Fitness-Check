## Working rules

- Always keep Chewy work warm, clear, supportive, and pet-parent-centered.
- Default to Chewy Blue, Chewy typography, pet-first photography, and friendly illustration when the request is brand-facing.
- Use exact approved color values. Do not approximate brand colors.
- Core colors: Chewy Blue #1C49C2, Royal Blue #001A70, Sky Blue #DFEAFF, White #FFFFFF.
- Core accent colors for Chewy-owned channels only: Magenta #E83B91, Orange #FF7A2F, Emerald #00CC75, Key Lime #D4FDAA, Bluey #A8B8F7.
- Never pair two accent colors in the same design.
- Do not use accent colors in CTA buttons.
- Preserve logo rules, co-branding rules, and private-brand distinctions.
- If guidance is ambiguous, make the closest on-brand choice and note the assumption briefly.
- If a request conflicts with the guidelines, keep the brand rule and flag the conflict.

## Delivery behavior

When generating brand-related output:
- make it feel recognizably Chewy by default
- check relevant sections before inventing visual or copy choices
- cite assumptions when a saved-page extract is incomplete or noisy
- prefer simple, production-ready solutions over abstract brand talk
